# Проверенные источники заказов

Основной источник сейчас не Telegram, а веб-лента Kwork:

- `https://kwork.ru/projects?c=11` — свежие проекты категории "Разработка и IT". Парсер берет только карточки с видимым счетчиком `Предложений` и дедуплицирует их по project id.

Telegram оставлен как запасной вариант для public read-only режима без Telegram API:

- `@freelance_dev_work` — Kwork-проекты по разработке. В постах есть кнопка "Связаться с заказчиком"; парсер вытаскивает ее как ссылку для отклика. Самый полезный источник для простых WordPress/HTML/CSS/JS правок.

Сейчас проект переведен в Kwork web-only режим: остальные источники не используются, чтобы не мешать быстрой реакции на новые Kwork-заказы.

Проверенные, но пока не добавленные:

- `@FreelancehuntProjects` — свежие проекты Freelancehunt. В постах есть кнопка "Зробити ставку"; подходит для задач по верстке, WordPress и ускорению сайтов.
- `@freelance_wordpress` — подборка заказов для WordPress-разработчиков с различных бирж. Целевые заказы по верстке, доработкам, плагинам.
- `@Easy_wrk` — Easy Work | Фриланс заказы | Удаленная работа. Разноплановые IT-заказы.
- `@freelance_work` — канал с проектами, заказами и удаленной работой на дому. Широкий спектр задач.
- `@digitaltender` — технически читается и имеет кнопки контакта, но в последних постах мало подходящих простых web-заказов.
- `@FreeVacanciesIT`, `@webfrl`, `@frilans` — много вакансий, резюме, дизайна и нецелевого шума, поэтому не включены в дефолтную подборку.

Текущий дефолт в `.env.example`:

```text
KWORK_SOURCE=web
KWORK_PROJECTS_URL=https://kwork.ru/projects?c=11
TELEGRAM_CHANNELS=@freelance_dev_work
```
